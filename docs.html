SimplyFive — One-click LOD GeneratorSimplyFive — генератор LOD в один клик

A Blender add-on that generates a set of LOD meshes (Levels of Detail) from the active object, using the meshoptimizer library. For each mesh you can create up to 5 simplified versions with individual aggressiveness; simplification can take UV unwrapping, normals, vertex colors (as an importance map) and materials into account.Аддон для Blender, который генерирует набор LOD-мешей (Level of Detail) из активного объекта с помощью библиотеки meshoptimizer. Для каждого меша можно создать до 5 упрощённых версий с индивидуальной агрессивностью; упрощение может учитывать UV-развёртку, нормали, вершинные цвета (как карту важности) и материалы.

1.What the add-on doesЧто делает аддон

SimplyFive creates several simplified copies of your model in one click (LOD — Level of Detail): the farther an object is from the camera, the fewer polygons it needs. The add-on figures out on its own how to remove the “extra” triangles so that the silhouette, textures and shading suffer as little as possible.

SimplyFive создаёт несколько упрощённых копий модели в один клик (LOD — уровень детализации): чем дальше объект от камеры, тем меньше полигонов ему нужно. Аддон сам решает, как убрать «лишние» треугольники, чтобы силуэт, текстуры и затенение пострадали как можно меньше.

2.Installing the add-onУстановка аддона

  1. Open Blender → Edit → Preferences → Add-ons.Откройте Blender → Edit → Preferences → Add-ons.
  2. Click “Install...” and select the SimplyFive folder/archive.Нажмите «Install...» и выберите папку/архив SimplyFive.
  3. Enable the checkbox next to the add-on in the list.Включите галочку напротив аддона в списке.

2.1 The meshoptimizer library2.1 Библиотека meshoptimizer

The add-on uses the third-party meshoptimizer library, which does the actual geometry simplification. It is already bundled inside the add-on — no extra steps are needed, the generation buttons work right after installation. You can open Preferences to confirm the library status reads ready.

Аддон использует стороннюю библиотеку meshoptimizer, которая выполняет само упрощение геометрии. Она уже встроена в аддон — ничего дополнительно делать не нужно, кнопки генерации работают сразу после установки. В Preferences можно убедиться, что статус библиотеки — ready (готова).

Pro Tip: Pro can also rebuild the library from source (e.g. to pull a newer meshoptimizer) with a button in Preferences — see Appendix A. Light always ships and loads the bundled library only.Совет: Pro также умеет пересобирать библиотеку из исходников (например, чтобы взять более свежий meshoptimizer) кнопкой в Preferences — см. Приложение A. Light всегда только загружает встроенную библиотеку.

3.Where to find the add-on panelГде находится панель аддона

  1. Select the mesh object you want to simplify in the 3D viewport.Выделите меш-объект для упрощения во вьюпорте.
  2. Open the sidebar with the N key.Откройте боковую панель клавишей N.
  3. Find the “LODs” tab on the right (the Pro tab is “LODs”; the Light tab is “LODsLight”).Найдите вкладку «LODs» справа (в Pro вкладка «LODs»; в Light — «LODsLight»).

At the top of the panel you’ll see the selected object’s name and its current triangle / vertex count.

Вверху панели — имя выбранного объекта и его текущее число треугольников / вершин.

4.Creating LODs in 5 stepsСоздание LOD в 5 шагов

  1. Select the model (best to work with final, finished geometry).Выделите модель (лучше работать с финальной геометрией).
  2. In “Number of LODs”, set how many levels you need (3–4 is usually enough).В поле «Number of LODs» задайте число уровней (обычно достаточно 3–4).
  3. For each level, pick a Mode from the dropdown: the farther the level is from “0”, the more aggressive the mode should be (see the table in section 5).Для каждого уровня выберите режим (Mode): чем дальше уровень от «0», тем агрессивнее режим (см. таблицу в разделе 5).
  4. Optionally adjust the triangle percentage (%) for each level.При желании поправьте процент треугольников (%) для каждого уровня.
  5. Click the big “Generate LODs” button at the bottom.Нажмите большую кнопку «Generate LODs» внизу.

After generation, new objects appear: your original is renamed to name_lod_0, and name_lod_1, name_lod_2, … appear next to it — one per level.

После генерации появятся новые объекты: исходный переименуется в name_lod_0, рядом появятся name_lod_1, name_lod_2, … — по одному на уровень.

Important:Важно: Running generation again for the same object deletes and recreates old LOD objects with the same names — previous versions are not kept.Повторный запуск генерации для того же объекта удаляет и пересоздаёт старые LOD-объекты с теми же именами — предыдущие версии не сохраняются.

6.1 Vertex colors as an “importance map”6.1 Вершинные цвета как «карта важности»

If important areas are painted white (in Vertex Paint mode) and unimportant ones black, the add-on can treat important zones (a face, a logo, a small detail) more carefully and simplify unimportant ones (a back, a sole) more aggressively.

Если важные области покрасить белым (в режиме Vertex Paint), а неважные — чёрным, аддон будет бережнее относиться к важным зонам (лицо, логотип, мелкая деталь) и агрессивнее упрощать неважные (спина, подошва).

OptionОпцияWhat it doesЧто делает
Use Vertex Color as ImportanceTurns the idea on. Reads the active vertex color layer as a per-vertex importance map (luminance: white = important). Applies to every LOD.Включает механику. Читает активный слой цвета вершин как карту важности (яркость: белый = важно). Действует на все LOD.
Importance StrengthHow strongly the painting biases the result (0–1). A soft weight: higher = more careful around white zones, but not a full guarantee even at maximum.Насколько сильно покраска влияет на результат (0–1). Мягкий вес: выше = бережнее к белым зонам, но не полная гарантия даже на максимуме.
Hard-Lock Above Threshold ProFor an actual guarantee: vertices whose importance is above the threshold below are fully protected from collapsing (same hard-lock mechanism as Protect UV Seams).Для настоящей гарантии: вершины с важностью выше порога ниже полностью защищены от схлопывания (тот же механизм, что Protect UV Seams).
Lock Threshold ProThe brightness threshold (0–1) for the hard protection above. E.g. 0.9 = only the brightest painted zones get hard-protected.Порог яркости (0–1) для жёсткой защиты выше. Напр. 0.9 = защищаются только самые яркие зоны.

7.Fine-tuning each level — the “Details” sectionТонкая настройка каждого уровня — раздел «Details» Pro

The built-in modes are enough for most cases. But if something’s off — the model “breaks”, the texture drifts, or too many polygons remain — expand “Details” under that level’s mode and adjust the parameters by hand. Changing any of them switches the mode to “Custom” automatically.

Встроенных режимов хватает в большинстве случаев. Но если что-то не так — модель «ломается», текстура плывёт или остаётся слишком много полигонов — разверните «Details» под режимом уровня и правьте параметры вручную. Любое изменение автоматически переключает режим в «Custom».

7.1 Target Error — the main parameter7.1 Target Error — главный параметр

The “allowed error” of simplification — how much the result may differ from the original (by default a fraction of the model’s size, 0–1).

«Допустимая ошибка» упрощения — насколько результат может отличаться от оригинала (по умолчанию доля размера модели, 0–1).

  • Lower → stops earlier and more cautiously; closer to the original, but more triangles than the target % may remain.Меньше → останавливается раньше и осторожнее; ближе к оригиналу, но может остаться больше треугольников, чем задано в %.
  • Higher → allowed to cut harder to reach the target %, at the cost of more visible distortion.Больше → разрешено резать сильнее ради целевого %, ценой более заметных искажений.
Important:Важно: Too high a Target Error can collapse the mesh completely (especially with a low %). If the object caves in or disappears — lower Target Error first.Слишком большой Target Error может полностью схлопнуть меш (особенно при низком %). Если объект проваливается или исчезает — сначала уменьшите Target Error.

7.3 Textures (UV) and shading (normals)7.3 Текстуры (UV) и затенение (нормали)

ParameterПараметрWhat it doesЧто делает
Permissive (aggressive)Lets simplification jump over texture seams if it greatly cuts polygons. For far, rough LODs; not for close-ups.Позволяет упрощению «перепрыгивать» через UV-швы, если это сильно снижает полигоны. Для дальних грубых LOD; не для крупных планов.
Protect UV SeamsOnly with Permissive: protects seams where the texture clearly differs on either side, cutting “invisible” seams freely.Только с Permissive: защищает швы, где текстура явно различается по сторонам, свободно срезая «невидимые» швы.
Preserve UVs & NormalsThe main “smart” toggle: tries not to ruin texture and shading. Turn off only for the farthest/roughest LODs.Главный «умный» переключатель: старается не портить текстуру и затенение. Выключайте только для самых дальних/грубых LOD.
Normal WeightHow strongly shading is protected (0–1). Increase if you see creases/odd highlights; decrease to push polygons lower.Насколько сильно защищается затенение (0–1). Повышайте при заломах/бликах; понижайте, чтобы срезать больше полигонов.
UV WeightSame idea for texture coordinates. Increase if the texture drifts/stretches; decrease for far LODs.То же для текстурных координат. Повышайте, если текстура плывёт/тянется; понижайте для дальних LOD.
Vertex UpdateSlightly moves vertices and texture to fit the new shape. Better results at aggressive levels (LOD3+).Слегка двигает вершины и текстуру под новую форму. Лучше результат на агрессивных уровнях (LOD3+).

ParameterПараметрWhat it doesЧто делает
Pre-pruneA separate pass BEFORE main simplification (threshold 0–0.2) that strips small debris without affecting Target Error accuracy. Start at 0.01–0.05.Отдельный проход ДО основного упрощения (порог 0–0.2), срезающий мелкий мусор без влияния на точность Target Error. Начните с 0.01–0.05.

8.Checking the resultКак проверить результат

After generation, every level sits stacked in the scene. To evaluate a specific level:

После генерации все уровни лежат стопкой в сцене. Чтобы оценить конкретный:

  • Move the “LOD Preview (distance)” slider at the top — 0 shows the original, higher values show the corresponding simplified level (others hide automatically).Двигайте слайдер «LOD Preview (distance)» вверху — 0 показывает оригинал, выше — соответствующий упрощённый уровень (остальные скрываются).
  • Or click “Only This LOD” next to the level you want; “Show All LODs” makes every level visible again.Или нажмите «Only This LOD» у нужного уровня; «Show All LODs» снова показывает все.
  • The “Line Up LODs” button lays every level out in a row (isolated in local view) to compare the progression side by side.Кнопка «Line Up LODs» выстраивает все уровни в ряд (изолируя в local view) для сравнения бок о бок.

9.Fixing just one levelИсправить только один уровень

You don’t have to recreate everything. Change the settings for the level you want and click “Generate This LOD” in its block — the other levels stay untouched.

Не обязательно пересоздавать всё. Измените настройки нужного уровня и нажмите «Generate This LOD» в его блоке — остальные уровни не тронутся.

10.Saving your own settingsСохранение своих настроек Pro

If you’ve worked out a configuration you like (number of levels, percentages, modes, and every Details value), save it as a preset and apply it to other models in one click:

Если вы подобрали удобную конфигурацию (число уровней, проценты, режимы и все значения Details), сохраните её как пресет и применяйте к другим моделям в один клик:

  1. Set up all the LOD levels the way you want.Настройте все уровни LOD как нужно.
  2. Click the “+” next to the “LOD Presets” dropdown and name the preset.Нажмите «+» рядом со списком «LOD Presets» и назовите пресет.
  3. Next time, just pick this preset from the list — every setting is applied instantly.В следующий раз просто выберите пресет из списка — все настройки применятся мгновенно.

The “Restore Defaults” button in the same menu resets settings to factory values.

Кнопка «Restore Defaults» в том же меню сбрасывает настройки к заводским.

The library is already bundled, so this isn’t needed for normal use. Rebuild only if you want the very latest meshoptimizer from GitHub, or the bundled library is damaged.

Библиотека уже встроена, для обычной работы это не нужно. Пересобирайте, только если хотите самый свежий meshoptimizer с GitHub или встроенная библиотека повреждена.

Important:Важно: Building requires internet access and a C/C++ compiler installed (the same one used to build other C add-ons).Для сборки нужен интернет и установленный компилятор C/C++ (тот же, что для других C-аддонов).
  1. Open Edit → Preferences → Add-ons → SimplyFive and expand its settings.Откройте Edit → Preferences → Add-ons → SimplyFive и разверните настройки.
  2. Click “Build meshoptimizer (one-time)” (or “Rebuild”).Нажмите «Build meshoptimizer (one-time)» (или «Rebuild»).
  3. Wait ~a minute — the add-on downloads the source, compiles it, and loads the library.Подождите ~минуту — аддон скачает исходники, скомпилирует и загрузит библиотеку.
  4. Confirm the status changed to the “ready / built” state.Убедитесь, что статус сменился на «готово / built».

If something goes wrong, the error text is shown there and in Blender’s system console (Window → Toggle System Console on Windows) — usually a missing compiler or no internet.

Если что-то пошло не так, текст ошибки — там же и в системной консоли Blender (Window → Toggle System Console на Windows) — обычно нет компилятора или интернета.
