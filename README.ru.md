# SimplyFive Light — генератор LOD для Blender в один клик

Аддон для Blender, который генерирует набор LOD-мешей (Level of Detail) из
активного объекта. Для каждого меша можно создать до 5 упрощённых версий,
каждая со своим процентом треугольников и режимом качества — в один клик.

**Light** — бесплатная версия. Она самодостаточна и полезна сама по себе;
отдельная версия **Pro** добавляет тонкую настройку каждого уровня и
дополнительные опции переноса (см. раздел *Версия Pro* ниже).

Автор: zloy_pingvin
Требуемая версия Blender: 5.0.0+ Возможна работа на версиях 3.0.0+ но не была достаточно оттестирована

## Возможности (Light)

- До 5 LOD-уровней на объект, каждый со своим процентом треугольников.
- 4 режима качества в один клик на уровень: **Careful**, **Standard**,
  **Aggressive**, **Very Aggressive** — чем дальше уровень, тем агрессивнее
  режим.
- Атрибутно-осознанное упрощение: сохраняет UV и нормали, чтобы швы и жёсткие
  рёбра не разрушались.
- Вершинные цвета переносятся на LOD и, по желанию, используются как карту
  важности по вершинам (закрасьте области, где нужно сохранить больше
  деталей) — один общий переключатель на все уровни.
- Поддержка нескольких материалов, включая материалы на линкованных
  дубликатах/инстансах или полученные через Geometry Nodes.
- Merge by Distance на результате для чистого сваривания швов.
- Настраиваемый суффикс имени LOD (Preferences), по умолчанию `_lod_`.
- Предпросмотр LOD прямо во вьюпорте, а также режим обзора **Line Up
  LODs** — выстраивает все уровни в ряд для сравнения бок о бок.
- Перенос нескольких UV-каналов

## Версия Pro https://zloy-pingvin.github.io/SimplyFive-Light/

Pro включает всё из Light и добавляет:

- Полную тонкую настройку каждого параметра упрощения для каждого уровня.
- Цепочку LOD — упрощение каждого уровня из предыдущего для более чистых
  дальних LOD.
- **Жёсткую блокировку** по цвету вершин: гарантия, что важные области не
  будут упрощены.
- Пересчёт нормалей + Auto Smooth по углу для очень низкополигональных
  уровней.
- Дополнительный ультра-агрессивный режим, игнорирующий топологию, и проход
  «регуляризации» для более равномерных треугольников.
- Сохранение и загрузку полной конфигурации в виде именованных пресетов.


## Установка

1. Blender → Edit → Preferences → Add-ons.
2. «Install...» → выбрать папку/архив SimplyFive Light.
3. Включить галочку напротив аддона «SimplyFive Light (lod generator)».

Библиотека упрощения уже встроена в аддон — ничего дополнительно собирать не
нужно.

## Быстрый старт

1. Выделите меш-объект во вьюпорте.
2. Откройте боковую панель (N) → вкладка «LODs».
3. Задайте «Number of LODs» (обычно достаточно 3–4).
4. Для каждого уровня выберите режим (Careful / Standard / Aggressive /
   Very Aggressive) — чем дальше уровень от «0», тем агрессивнее должен быть
   режим.
5. При желании поправьте процент треугольников (%) для каждого уровня.
6. Нажмите «Generate LODs».

Исходный объект будет переименован в «имя_lod_0», рядом появятся
«имя_lod_1», «имя_lod_2» и так далее — по одному на каждый уровень (суффикс
`_lod_` можно поменять в Preferences). Слайдер «LOD Preview (distance)»
позволяет посмотреть каждый уровень отдельно, а кнопка-значок рядом с ним
(«Line Up LODs») выстраивает все сгенерированные уровни в ряд для сравнения
бок о бок.

полная документация https://zloy-pingvin.github.io/SimplyFive-Light/docs.html
